3. 기본으로 생성되는 master 브랜치를 main브랜치로 변경
	git chackout -b main : main 브랜치를 생성하고 해당 브랜치로 이동
	git branch -M main : 기본 브랜치를 main 브랜치로 지정
	git status : 상태 확인
